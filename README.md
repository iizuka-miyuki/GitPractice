# 概要
Gitの使い方を学ぶ為に作成したリポジトリです。手を動かしてgitの使い方を学習していきましょう。  
# 使い方
①利用する際はForkしてから利用すること。  
②クローンをして、practiceディレクトリ内にあるものを進めていくこと。  
③ファイルごとに作業ブランチをmasterブランチから切り、PR（プルリクエスト)を作成してから、masterブランチにマージすること。
